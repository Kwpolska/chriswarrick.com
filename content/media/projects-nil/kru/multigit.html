Updating all at once. ============== PURPOSE ------- These tools run specific git commands. The script requires a special dir for repos. (~/git by default) INSTALLATION ------------ Copy it to /usr/bin or ~/bin. You may need to change the interpreter. NOTES ----- gitcommiter passes all arguments to "git commit" commands.

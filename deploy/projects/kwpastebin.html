KwPastebin. The easiest pastebin ever made.

PURPOSE

Pastebin. Simple pastebin.

INSTALLATION

  1. Put the files on your server.
  2. Edit the config.php.
  3. Execute the install.php file (you can do it form the command line, too).
  4. That’s it!

UPGRADE

Before upgrading the script you shall read the COMPATIBILITY.md file. It will inform you about all the major changes. This project is in development all the time. The database or behavior might be changed in a bigger way. A note to COMPATIBILITY.md will be added with each major upgrade.

The COMPATIBILITY.md file is available in the github repository. You can also find it at https://github.com/Kwpolska/kwpastebin/raw/master/COMPATIBILITY.md or (if you like to read nice HTML) https://github.com/Kwpolska/kwpastebin/blob/master/COMPATIBILITY.md.

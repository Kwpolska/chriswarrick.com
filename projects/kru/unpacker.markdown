## Description

Unpacker is a llttle utility made for unpacking various archives. Supported archives:

* tar.gz/tgz
* tar.bz2/tbz2
* tar.lzma
* tar.xz
* lzma
* xz
* bz2
* rar
* gz
* tar
* zip
* Z
* 7z
* cab

au­rqt is a desk­top graph­i­cal in­ter­face to the AU­R. It lever­ages the pow­er of the AUR API and the Web in­ter­face.

au­rqt de­pends on PKG­BUILDer for most fea­tures.

au­rqt should be in­stalled from the AUR it­self, by in­stalling the au­rqt pack­age:

pkgbuilder -S aurqt

The project is par­tial­ly aban­doned and may not work prop­er­ly.

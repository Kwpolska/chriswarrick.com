Status Production/Stable

A simple date calculator that can calculate the difference between two dates or add/subtract a given number of days, hours, minutes, seconds to a date.

User interfaces: CLI (datecalc, datecalc-cli) and GUI (datecalc-gui — requires PyQt5).
